Objetivos del OVI

  • Comprender el porque debe de hacerse el desarrollo de un software de esta manera.
  • Como obtener los requerimientos necesitados por la persona que solicita el proyecto.
  • Realizar un analisis eficiente sobre los requerimientos.
  • Aprender a realizar el diseño sobre los requerimientos.
  • Comprender cada una de las etapas del diseño de software y su estructura.
  • Manejar los modelos de desarrollo de software.
  • Conocer cada uno de los modelos.
  • Aprender que modelo se utiliza en algun caso especifico.

Objetivos del curso

  • Desarrollar paginas web con el lenguaje de marca o etiquetas html5 y realizar los diseños con la hoja en cascada css3.
  • Comprender cada una de las etiquetas que hacen parte de html5 y el desarrollo de cada una asi como su funcionalidad.
  • Diseñar la pagina web a base de comandos que hacen parte de css3 para una buena visualización.
  • Las paginas web se deben poder visualizar en cualquier navegador web.

Tematica a tratar

  • Etapas del desarrollo de software.
    • Requerimientos.
    • Analisis.
    • Diseño.
    • Codificación.
    • Implementación.
  • Modelos de desarrollo de software
    • Modelo de cascada.
    • Modelo de espiral.
    • Desarrollo iterativo e incremental.
    • Desarrollo agil.
    • Orientado a la reutilización.
